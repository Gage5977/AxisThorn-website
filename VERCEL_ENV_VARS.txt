HOW TO ADD IN VERCEL:

1. Go to: https://vercel.com/axis-thorns-projects/axis-thorn-llc-website/settings/environment-variables

2. For each variable above:
   - Click "Add New"
   - Key: Copy the variable name (e.g., JWT_SECRET)
   - Value: Copy the value after the = sign
   - Environment: Select all (Production, Preview, Development)
   - Click "Save"

3. After adding all variables, redeploy:
   - Go to Deployments tab
   - Click "..." on the latest deployment
   - Select "Redeploy"
